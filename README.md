# CS-LightGBM-Heart-Disease
A cost-sensitive LightGBM model for early heart disease prediction using SHAP explainability.

## Dataset
UCI Heart Disease Dataset — https://archive.ics.uci.edu/dataset/45/heart+disease

## Model
CS-LightGBM — Modified LightGBM with cost-sensitive class penalty weights
